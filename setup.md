# Setup Guide
## Project: Sands Of The Sphinx

---

# 1. Overview

This guide explains how to install and configure the required software for running Sands Of The Sphinx.

The project uses:
- Unity
- Ollama
- Gemma 3:4B
- Local AI inference

The game generates Egyptian-themed riddles locally using AI.

---

# 2. Required Software

## Unity

Recommended version:
- Unity 2022 LTS or newer

Download:
https://unity.com/download

Recommended modules:
- Windows Build Support
- TextMeshPro

---

## Ollama

Ollama is used to run local language models.

Download:
https://ollama.com/download

Supported operating systems:
- Windows
- Linux
- macOS

---

# 4. Installing Ollama

## Step 1 - Download Ollama

Visit:

https://ollama.com/download

Download the correct installer for your operating system.

---

## Step 2 - Install Ollama

Run the installer and follow the installation instructions.

After installation, verify it works by opening a terminal and running:

```bash
ollama --version
